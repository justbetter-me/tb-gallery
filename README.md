# tb-agent — discovered tight-binding Hamiltonians (gallery)

Live gallery of symbolic tight-binding Hamiltonians **H(k)** discovered for 2D materials, each
reproducing its SIESTA DFT bands. Open **index.html** (or the GitHub Pages link) and select a
material to see its symmetry, the DFT-vs-TB band overlay, the computed scorecard, the H(k) formula,
and the investigation notes.

Generated from the (private) `tb-agent` research repo via `scripts/build_gallery.py`.
